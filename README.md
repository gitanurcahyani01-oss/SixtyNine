# SixtNine — Sneaker Website

## Struktur Folder
```
sixtnine/
├── index.html          ← Halaman utama
├── css/
│   └── style.css       ← Semua styling
├── js/
│   └── main.js         ← Interaktivitas (cart, modal, nav, dll)
└── img/
    ├── placeholder.svg ← Otomatis dipakai jika gambar tidak ada
    ├── hero-shoe.png   ← Gambar sepatu hero section (atas besar)
    ├── shoe-1.png      ← Agile Blue Runner
    ├── shoe-2.png      ← Urban Chill Sneaker
    ├── shoe-3.png      ← Pace Trainer
    ├── shoe-4.png      ← Street Flex Pro
    ├── shoe-5.png      ← Cloud Walker
    ├── shoe-6.png      ← Neon Dash X
    ├── arrival-shoe.png← Gambar di banner New Arrivals
    └── about-shoe.png  ← Gambar di section About

```

## Cara Pakai
1. Letakkan gambar-gambar sepatu di folder `img/`
2. Sesuaikan nama file sesuai daftar di atas
3. Buka `index.html` di browser

> **Catatan:** Jika gambar tidak tersedia, otomatis akan tampil placeholder SVG biru.
